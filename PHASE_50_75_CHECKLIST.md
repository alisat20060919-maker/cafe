# 50～75 步驗收清單

這份只列 50～75 步的功能驗收。每次改完核心玩法後，用最新 `?force=coreXX` 網址在手機瀏覽器跑一次。

## 50 MVP 核心循環

1. 開首頁不白屏。
2. 後山採集 1 次，背包增加素材。
3. 溫室採集 1 次，背包增加素材。
4. 廚房製作 1 個委託成品。
5. 委託頁交付成品，扣掉 requiredItems。
6. 交付後發放葉幣、星糖與 EXP。
7. 重新整理頁面後，獎勵與完成狀態仍保留。

## 60 等級鎖與升級鏈

1. 初始 Lv.1，煉金室維持鎖定。
2. 完成足夠委託取得 EXP。
3. 升到 Lv.2 時彈出 Level Up Modal。
4. Level Up Modal 顯示新解鎖內容。
5. 關閉彈窗後，煉金室變成可進入。
6. 舊存檔開啟後仍會自動套用等級解鎖。

## 70 精靈與祈願鏈

1. 祈願 1 次會先顯示祈願演出，再顯示結果。
2. 星糖不足時不扣資源，顯示失敗提示。
3. 抽到 item 會進背包。
4. 抽到 fairy 會進 state.fairies，精靈頁顯示已契約。
5. 送禮會扣除背包禮物並增加好感度。
6. 精靈被動 Buff 會影響對應採集或委託數值。
7. 祈願紀錄會保留最近 20 筆。
8. 保底觸發時結果卡與演出有特殊提示。

## 75 委託刷新

1. 委託頁顯示今日委託刷新列。
2. 免費刷新可使用一次。
3. 用過免費刷新後，免費刷新按鈕 disabled。
4. 付費刷新會扣靈感券。
5. 星糖重抽會扣星糖。
6. 資源不足時不刷新，顯示失敗提示。
7. 刷新後每日委託列表重新洗牌。
8. 刷新不應把特殊任務或精靈委託塞入 daily ids。
9. 刷新後仍可正常切換每日任務 / 特殊任務 / 精靈委託。

## UI 補完檢查

1. 未解鎖委託有 LOCKED 視覺與條件提示。
2. 商店可從右上商店圖示進入。
3. 商店可從咖啡廳場景「商鋪」入口進入。
4. 祈願演出不造成白屏。
5. 委託刷新、交付、切分類都使用 event delegation，不重複綁定大量 listener。
