# 精靈咖啡屋 Cafe

這是一個用來整理「精靈咖啡屋」世界觀、菜單、素材與委託商品的網站 repo。

## 目前內容

- `index.html`：網站首頁
- `style.css`：首頁樣式

## 網站方向

精靈咖啡屋是一間販售魔法飲品、甜點、素材與小物的沉浸式幻想店鋪。這個網站之後可以慢慢擴充成：

- 菜單頁
- 素材圖鑑
- 角色介紹
- 委託品項說明
- 世界觀設定頁

## 下一步可新增

- `menu.html`：正式菜單
- `items.html`：素材圖鑑
- `characters.html`：店員與精靈介紹
- `commissions.html`：沉浸式委託說明
